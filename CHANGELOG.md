# Changelog
All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-14
### Added
- 🎉 Initial release of FIA-X (ai-fia) AI-driven innovation framework
- ✅ 三大认知支柱：First-principle + Interdisciplinary + Anti-consensus
- ✅ 十步标准化创新流程：从问题拆解到持续进化的全闭环
- ✅ 3个核心工具脚本：
  - `fia-innovation-assistant.py`: 交互式创新流程助手
  - `innovation-evaluation-tool.py`: 创新方案量化评估工具
  - `anti-consensus-mining-tool.py`: 反共识机会挖掘工具
- ✅ 完整知识库：
  - 9个精选创新案例（技术/产品/反共识）
  - 3大行业共识库（医疗AI/新能源/互联网）
  - 6大类近100个跨学科思维模型
  - 4份标准化模板（十步流程/三大支柱模板）
- ✅ 全符合OpenClaw技能规范，开箱即用
- ✅ 国际化命名 `ai-fia` (AI First Innovation Framework)，适合全球传播
- ✅ 完整文档：readme.md、install.md、extension-spec.md

### Features
- 支持3种触发方式：自然语言、命令行、API调用
- 完全通用，适用于所有行业的创新场景
- 支持行业扩展，无需修改核心代码
- 量化评估体系，创新能力可衡量、可对比
- 创造力数学模型，实现理论到实践的闭环
